# 🧠 AI Productivity Predictor

## 🚀 Overview
This project predicts a user's productivity score based on lifestyle factors such as screen time, study hours, sleep, and focus level.

---

## 🔥 Features
- Machine Learning model (Random Forest)
- FastAPI backend
- React frontend
- Real-time prediction system

---

## 🛠 Tech Stack
- Python (pandas, scikit-learn)
- FastAPI
- React.js
- Axios
- Joblib

---

## ▶️ How to Run

### 🔹 Backend
cd backend
uvicorn app:app --reload


### 🔹 Frontend
cd frontend
npm install
npm start


---

## 📊 Example Input
age: 22
daily_screen_time: 5
social_media_hours: 2
study_hours: 4
sleep_hours: 7
notifications_per_day: 100
focus_score: 80
addiction_level: 1
